# Flutter Snap!

A new Flutter project.


## app icons

```
flutter pub run flutter_launcher_icons
```

## android build

```
flutter build appbundle  --build-name "1.0" --build-number 1
flutter install
```


## firebase functions

inside firebase-functions run `npm run deploy`
use emulators `npm run serve`
